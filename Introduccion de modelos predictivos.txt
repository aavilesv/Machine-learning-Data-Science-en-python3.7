Modelos predictivos
Este tipo de análisis utiliza métodos matemáticos para pronosticar eventos o resultados futuros. Mediante un proceso iterativo, se desarrolla el modelo predictivo mediante un conjunto de datos de entrenamiento y después se prueba y se valida para determinar su precisión con el fin de realizar los mejores pronóstico
